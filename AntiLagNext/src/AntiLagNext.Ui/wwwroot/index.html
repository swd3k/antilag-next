Перезагрузить ПК?

Оптимизация включена. Таймер и схема питания уже активны. Перезагрузка желательна, чтобы часть твиков реестра и служб применилась полностью — не обязательна сразу.

Также включены: сворачивание в трей и авто-оптимизация при следующих стартах.

Запуск с Windows?

Будет создана задача Планировщика (ONLOGON, highest privileges). При входе в систему AntiLag Next сможет стартовать с правами администратора без UAC каждый раз.

Можно отключить позже в «Настройки». Без подтверждения задача не создаётся.

Перед включением

  1. Что изменится: timer resolution, схема питания, часть реестра GPU/Game Mode — с JSON-бэкапом.
  2. Откат: кнопка «Сбросить всё» / «Отключить» восстанавливает сохранённое состояние.
  3. График µs: proxy scheduling latency, не kernel DPC и не network ping.
